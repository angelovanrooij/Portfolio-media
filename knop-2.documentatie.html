Media Periode 2 (Maatwerk)



Knop 2


Tijdens deze periode heb ik aan een opdracht gewerkt in de vorm van een knop

Voordat ik het eindproduct heb gemaakt, heb ik voorafgaand nagedacht over hoe en wat ik wilde gaan maken

Hiervoor heb ik andere opdrachten gemaakt. Deze zijn hieronder te vinden.


Tijdens de eerste opdracht heb ik inspiratie opgedaan door 2 aan willekeurige voorwerpen te denken en hiervan de interface te verwisselen.
Ik heb gekozen voor een deobus en een camera. Het idee is om Foto's te kunnen maken van een apparaat dat lijkt op een deobus.
De deobus kun je draaien zoals een camera dat kan om te zoomen. Verder werkt het spuitgat zoals de sluiter van een camera.
Hiervan kun je instellingen aanpassen (sluitertijd etc.).



Vervolgens ben ik naar verschillende vormen van buttons gaan zoeken.
Voornamelijk fysieke buttons want er zijn genoeg digitale buttons te vinden op het internet.



Daarna ben ik een prototype gaan maken waarop ik heb geprobeerd snel een versie te maken om alvast een idee te krijgen.
Door dit prototype kon ik ook zien wat ik eventueel niet wilde.


Voor het onderdeel technisch ontwerp heb ik niet veel gezocht. Het is belangrijk om na te denken over wat je nodig hebt.
Omdat ik een website ga maken die gebruikt maakt van meerdere verschillende knoppen wilde ik sowieso HTML gebruiken.
Aangezien de website interactief moest zijn heb ik besloten gebruik te maken van javascript.
Echter, ik heb geen libraries gebruikt. Wel heb ik overwogen om JQuery te gebruiken. Dit heb ik uiteindelijk niet gedaan
Omdat dat alleen nodig zou zijn om de website "mooier" te maken, dus niet noodzakelik.



Ten slotte heb ik nagedacht over de user interface. Dit is belangrijk afhankelijk met welke doelgroep je te maken hebt.
Om hier goed over na te kunnen denken, heb ik verschillende schetsen gemaakt op een vel papier. Deze staat hieronder



Zelfreflectie


Tijdens het uitvoeren van dit project heb ik gemerkt dat het belangrijk is om van te voren goed na te denken.
Het scheelt namelijk veel tijd als je al weet hoe alles eruit moet gaan zien. Ik had missschien nog iets meerdere kunnen nadenken over thema's kleuren stijlen van buttons etc.
Toen alles af was vond ik het resultaat dan ook wat tegenvallen. Op het moment doe ik Software en Media en ben ik van plan om media te kiezen. De rede hiervoor is omdat
ik het leuk vond om hiermee bezig te zijn maar niet de tijdsdruk die erop zat. Ook gebrek aan kennis is jammer
Met software ben je vooral bezig met het functioneren van de je product terwijl je bij media meer nadenkt over
andere aspecten van een product zoals, doelgroep, interface, gebruikersvriendelijkheid.
Ik vind het persoonlijk leuker om me hiermee bezig te houden.


Na alle opdrachten ben ik de site gaan maken en is dit het resultaat...
